Table head options

Similar to tables and dark tables, use the modifier classes .thead-default or .thead-inverse to make <thead>s appear light or dark gray.

<table class="table">
  <thead class="thead-default">
    ...
  </thead>
  <tbody>
    ...
  </tbody>
</table>

<table class="table">
  <thead class="thead-inverse">
     ...
  </thead>
  <tbody>
     ...
  </tbody>
</table>

Striped rows

Add zebra-striping to any table row within the tbody.

<table class="table table-striped">
  ...
</table>

Bordered table

Tables with borders on all possible sides of the Table and Cells.

<table class="table table-bordered">
  ...
</table>

Hover Row

Enable a hover state on table rows within a tbody.

<table class="table table-hover">
  ...
</table>

Small table

Add .table-sm to make tables more compact by cutting cell padding in half.

<table class="table table-sm">
  ...
</table>
